
SHORTCODES

	Short code to show a single product with multple merchants and prices
		
		[affproduct product_id="169"]


	Short code to show multiple products and outputs the price from the cheapest merchant

		[affproducts product_ids="169, 154, 158" layout="list"] 

		// layout can be list or line

	Short code to show similar products

		[simproducts product_id="195" taxonomy="category" layout="line"] 

		// leave out taxonomy to use product categories
